Motor Control Center Documentation

I did a lot of field documentation while working here, and here is an example I did on a Motor Control Center. This was conducted by visiting the MCC rooms in our facility and opening up each motor control bucket we had, and documented the feeder configurations, what type of breaker it had, and other helpful speicfications to do a high level overview of the system to make it easy to know how each load was controlled, and where each load was controlled. Makes it a lot easier for electricians to debug as well as making it easier for the electrical engineers to plan future projects.

MCC Drawing 2

I did more documentation on nearly all of our I/O configurations of our entire PLC-5 Network as well as Controllogix. I did this through goung through our RSLogix5 software and noting down how each I/O was configured. Much of the data given on there was outdated or incomplete, so much of the work was also going to MCC rooms and confirming their configuration or to see if they were still in fact there. I recorded indiviual slot assignments and catalogged the input/output module types (IAD, OAD, OBD, IFE, OFE). I also made sure to document the processor's channel configuration which includes DH+ node addressing, dual RIO scanner channels serving six remote racks across two baud rates (115.2 kb and 230.4 kb), and serial channel settings.

Brine Pump Wiring Diagram

This drawing was for the Brine Pump, and shows how the 2 HP, 1750 RPM motor was configured and controlled. It shows which MCC room it is controlled, what frame it uses, and what NEMA size starter it uses. I also documented the type of overload it uses, the breaker and what it is set to, and the type of MCP.

The schematic shows the 480V/120V control circuit with PLC-5 integration and the addresses that it is linked to. Documented the elementary and wiring diagram for the R/O Brine Pump #1 (Drawing 13-3143, Released for Construction 03/16/12). The drawing covers a 2 HP, 1750 RPM, Frame 145T motor (Unit No. 2-734) housed in MCC Room 2 on a NEMA Size 1 starter with C356A overload, Size 3 breaker set at 10A, and an IECL36000 MCP. The PLC/DCS interface table identifies all signal types (Serial/DCS), addresses, and functions including Ready, Run Input, Run Output, Run Status, Start/Stop, and Pump Select.
